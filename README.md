# Movilidad Urbana y Productividad Económica en ciudades de LatinoAmerica

## Objetivo
Evaluar si la movilidad urbana tiene relación con la productividad económica en las principales ciudades de latinoamerica

## Datasets
- TomTom Traffic Index (datos de tráfico en tiempo real)
- OECD Cities (PIB per cápita, desempleo y población)

## Herramientas utilizadas
- Pyton
- Pandas
- Numpy
- Seaborn
- Matplotlib.pyplot

## Metología
1. Exploración de estructura y tipo de datos
2. Estandarización de nombres de columnas formato snake_case
3. Corrección de formato de datos y verificacion, para facilitar calculos matemáticos
4. Definición de termino a filtrar (año 2024)
5. Analisis y resumen de datasets separados
6. Union inner de ambos datasets para verificación de datos presentes en ambos
7. Visualización y analisis de relaciones

## Principales hallazgos
- Ciudades con un ingreso PIB per cápita no se relaciona directamente con una menor o mayor nivel de congestión
- Congestión vehicular parece estar influenciado por multiples factores además de la productividad económica
- Planificación urbana e infraestructura de transporte puede tener un rol importante en el desarrollo de la movilidad
